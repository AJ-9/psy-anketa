# Архивы проекта

Эта директория содержит архивы проекта для возможности отката к предыдущим версиям.

## Использование

### Создание архива вручную

```bash
npm run archive
```

или

```bash
node scripts/create-archive.js
```

### Восстановление из архива

```bash
# Распаковать архив
tar -xzf archives/psy-anketa_YYYYMMDD_HHMMSS.tar.gz -C /путь/куда/распаковать

# Или через Finder (macOS) - просто дважды кликните на архив
```

## Формат имени архива

Архивы именуются по шаблону: `psy-anketa_YYYY-MM-DDTHH-MM-SS.tar.gz`

Пример: `psy-anketa_2024-01-15T14-30-45.tar.gz`

## Что включается в архив

- Весь исходный код
- Конфигурационные файлы
- Документация

## Что НЕ включается

- `node_modules/` - устанавливаются через `npm install`
- `.next/` - генерируется при сборке
- `*.db` - база данных (создается автоматически)
- `.git/` - git репозиторий
- `archives/` - сама директория с архивами

## Автоматическое создание архивов

Архивы создаются автоматически при значительных изменениях в проекте.
