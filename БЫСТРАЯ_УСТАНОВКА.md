# ⚡ БЫСТРАЯ УСТАНОВКА Node.js

## Самый простой способ:

### 1. Откройте браузер и перейдите:
**https://nodejs.org/**

### 2. Нажмите большую зеленую кнопку:
**"Download Node.js (LTS)"**

### 3. Скачайте файл для macOS (`.pkg`)

### 4. Запустите скачанный файл и установите

### 5. **ВАЖНО:** Перезапустите терминал
- Закройте текущий терминал
- Откройте новый терминал

### 6. Проверьте установку:
```bash
node --version
npm --version
```

Должны показать версии (например: v20.x.x и 10.x.x)

### 7. После этого выполните:
```bash
cd /Users/alikbidzhiev/Documents/Cursor_Psy
npm install
npx prisma generate
npx prisma migrate dev --name init
npm run dev
```

---

**Это займет 2-3 минуты. После установки проект запустится!**
