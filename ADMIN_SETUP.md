# Настройка панели психолога

## Аутентификация

Панель психолога теперь защищена логином и паролем.

### Учетные данные по умолчанию:

- **Логин:** `admin`
- **Пароль:** `psy2024`

### Изменение учетных данных:

Для изменения логина и пароля создайте файл `.env.local` в корне проекта:

```env
ADMIN_USERNAME=ваш_логин
ADMIN_PASSWORD=ваш_пароль
```

**Важно:** В production используйте надежные пароли и храните их в секретах Vercel.

## Использование:

1. **Вход в панель:**
   - Перейдите на `/admin/login`
   - Введите логин и пароль
   - После успешного входа вы будете перенаправлены в панель

2. **Просмотр анкет:**
   - В панели психолога отображаются все сохраненные анкеты
   - Можно фильтровать по статусу (Все/Завершенные/Незавершенные)
   - Клик по анкете открывает детальную информацию

3. **Выход:**
   - Нажмите кнопку "Выйти" в правом верхнем углу панели

## Безопасность:

⚠️ **Для production рекомендуется:**

1. Использовать переменные окружения для хранения учетных данных
2. Реализовать более безопасную систему аутентификации (JWT с refresh tokens)
3. Добавить rate limiting для защиты от brute force
4. Использовать HTTPS для защиты cookies
5. Регулярно менять пароли

---

## API Endpoints:

- `POST /api/auth/login` - Вход в систему
- `GET /api/auth/check` - Проверка авторизации
- `POST /api/auth/logout` - Выход из системы
